# Advanced House Price Prediction using Regression Models

## Project Overview

This project demonstrates the implementation and comparison of advanced regression techniques for predicting house prices using Python and Scikit-learn. The project includes regularized linear models, cross-validation techniques, tree-based regression models, support vector regression, hyperparameter tuning, and comprehensive model evaluation.

The objective is to identify the most accurate regression model for predicting house prices while understanding the impact of regularization, validation strategies, and different machine learning algorithms.

---

## Dataset Information

The dataset contains approximately 3,800 house records with the following features:

- property_id
- sale_date
- area_sqft
- bedrooms
- bathrooms
- location_score
- property_age
- distance_city_km
- near_school
- near_metro
- crime_rate_index
- house_price_inr (Target Variable)
